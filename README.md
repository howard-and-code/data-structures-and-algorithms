# Data Structures And Algorithms

A web app to hold demonstrations of data structures and algorithms that are discssed on the Howard & Code [YouTube Channel](https://www.youtube.com/@howard-and-code).

## Dev Notes

### To create an Angular project at the root of the repo:

1. delete any files that are present (e.g. .gitignore, README.md)
2. run the command: ```ng new name-of-project --directory=.```
3. answer the yes/no questions; pick the CSS management you prefer
4. chug, chug, chug, ...
5. it's done
